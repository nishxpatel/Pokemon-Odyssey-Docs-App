# Discord Message

Hey! Just started Pokémon Odyssey and wanted a faster way to look up Pokémon, moves, items, and encounters instead of flipping through spreadsheets. So I built a docs site — think PokémonDB but custom-built for Odyssey.

**Features:**
✨ Searchable Pokédex with type filters & stat sorting  
⚡ Full 19×19 type chart (including Aether)  
🎯 Browse moves, abilities, items, and their sources  
📍 Wild encounters by location & stratum  
⭐ All Etrian Variants with extracted sprites  
🎨 Fully static, offline-friendly, no ads

**Live:** https://nishxpatel.github.io/Pokemon-Odyssey-Docs-App  
**GitHub:** https://github.com/nishxpatel/Pokemon-Odyssey-Docs-App

Built with a LOT of help from Claude (AI)—was my pair programmer the whole way. Open to feedback, bug reports, and feature ideas! 🙌
